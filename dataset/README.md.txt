# 🍅 Agro Tomato Quality Classifier

This project is a Deep Learning model to **automatically classify tomatoes** into three categories:
- ✅ Ripe
- 🟡 Unripe
- ❌ Damaged

## 📌 About the Project

In agriculture, especially in large-scale tomato farming, **manual quality inspection** is time-consuming and prone to errors. This project automates the classification of tomatoes using **image-based deep learning**.

## 🧠 Model Info

- 🔧 Model: Convolutional Neural Network (CNN)
- 📸 Input: Images of tomatoes (100+ images used)
- 🎯 Output: Class label (ripe / unripe / damaged)

## 📁 Dataset

The dataset includes 100 images, categorized into:
- `Ripe/`
- `Unripe/`
- `Damaged/`

📷 You can add more images to improve accuracy.

## 🛠️ Technologies Used

- Python 🐍
- TensorFlow / Keras
- NumPy, OpenCV
- Git & GitHub
